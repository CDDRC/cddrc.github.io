A demo project has been set up with permissions that will allow new users to explore a dataset in CDDRC Mosaic without having to set up their own project first. Follow the steps below to access and explore the demo project.

  1. Set up an account and login to the CDDRC Mosaic server

  2. Search for and/or navigate to the 468R-2018-gnomex_experiment-Yost-TRHTimecourse project:



  3. Open the 'Applications' page in the left navigation panel



  4. Click on the 'Gene Expression Explorer' application




  5. Explore the 'Gene Expression Explorer' application on the demo 468R project data: 1) modify the gene list and 2) modify the type of plot to display.




  6. Scroll further down the page to explore the 'Differential Expression Table'.
